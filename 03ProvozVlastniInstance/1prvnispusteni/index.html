Stažení zdrojových kódů a první spuštění

Tato sekce popisuje, jak zprovoznit program obsluhující vlastní instanci LingeBota.

Tento postup je proveden v příkazové řádce operačního systému Ubuntu 22. Předpokládá se, že je v systému nainstalován balík Python. Funkčnost programu je ověřena ve verzích 3.10 a 3.11 jazyka Python. Dalším předpokladem je, že pro bota máte založený účet a vygenerovaný token.

1. Naklonujte si GitHub repozitář LingeBota.

git clone https://github.com/RadekMocek/BP

2. Přejděte do adresáře BP/Bot/, kde se nachází zdrojový kód LingeBota.

cd BP/Bot/

3. Vytvořte konfigurační soubor config.json duplikováním souboru config-example.json.

cp config-example.json config.json

4. V konfiguračním souboru nastavte hodnotu záznamu tokenProduction na token vašeho bota a soubor uložte.

vim config.json

Poznámka: tokenProduction vs. tokenTesting

Pokud bude váš bot na produkci a zároveň do něj budete chtít přidávat a testovat na něm nové funkce, vyplatí se vytvořit si jeho druhou instanci. Token druhé instance pak nastavíte jako hodnotu záznamu tokenTesting a pomocí záznamu isProduction můžete snadno přepínat, zdali se spustí bot určený na produkci (true), nebo bot určený pro testování (false).

5. Nainstalujte potřebné Python balíčky.

pip3 install -r requirements.txt

6. Spusťte skript db_init.py pro vytvoření databáze.

python3 db_init.py

7. Spusťte bota spuštěním skriptu run.py. Pro budoucí spuštění stačí opakovat tento sedmý krok.

python3 run.py

8. Váš bot je nyní online.
