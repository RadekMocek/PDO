Vytvoření aplikace na Discord Developer Portal

Tato sekce popisuje, jak vytvořit aplikaci na Discord Developer Portal včetně jejího korektního nastavení pro provoz vlastní instance LingeBota, vygenerování zvacího odkazu a vygenerování tokenu.

1. Otevřete ve webovém prohlížeči stránku https://discord.com/developers/applications a klikněte na tlačítko New Application.

2. Do textového pole napište, jak chcete, aby se váš bot jmenoval. Poté zaškrtněte zaškrtávací pole pro souhlas s podmínkami. Nakonec potvrďte kliknutím na tlačítko Create.

3. Přejděte do sekce Installation kliknutím na příslušné tlačítko v levé navigační liště.

4. V části Install Link zvolte ve výběrovém seznamu možnost Discord Provided Link.

5. Do pole SCOPES v části Default Install Settings přidejte položku bot. Můžete buď napsat slovo bot jako u textového pole, nebo položku zvolit jako u výběrového seznamu.

6. Do nově objeveného pole PERMISSIONS přidejte tyto položky:

  • Attach Files
  • Embed Links
  • Manage Messages
  • Manage Roles
  • Read Message History
  • Read Messages/View Channels
  • Send Messages
  • Use Slash Commands

7. Uložte kliknutím na tlačítko Save Changes.

8. Z části Install Link si kliknutím na tlačítko Copy zkopírujte zvací odkaz vašeho bota a někam si ho uložte.

9. Přejděte do sekce Bot kliknutím na příslušné tlačítko v levé navigační liště.

10. Kliknutím na příslušný přepínač povolte možnost MESSAGE CONTENT INTENT.

11. Uložte kliknutím na tlačítko Save Changes.

12. V části TOKEN klikněte na tlačítko Reset Token, potvrďte tlačítkem Yes, do it!, poté zadejte své heslo a potvrďte tlačítkem Submit.

13. V části TOKEN se objevil token vašeho bota. Kliknutím na tlačítko Copy token zkopírujte a někam si ho uložte. Tento token uchovejte v tajnosti.

14. Bota nyní můžete přes jeho zvací odkaz pozvat na váš server stejným způsobem, jako jste pozvali hlavní instanci bota. Bot bude offline, protože zatím neběží jeho obslužný program.
