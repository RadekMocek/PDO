Spuštění LingeBota v prostředí Docker

Tato sekce popisuje, jak spustit LingeBota v prostředí Docker. Tento postup navazuje na postup stažení zdrojových kódů a prvního spuštění. Počítá se tedy s tím, že máte naklonovaný repozitář s botem a vyplněný konfigurační soubor config.json. Postup je proveden v operačním systému Ubuntu 22 s programem Docker ve verzi 26.0.

Upozornění: Vhodnost nasazení v prostředí Docker

Nasazení v prostředí Docker zlepší přenosnost řešení, ale zkomplikuje případné přidávání nového obsahu.

Použití Dockeru je vhodné pro dlouhodobé provozování bota, jehož obsah nechcete měnit.

Použití Dockeru (podle tohoto základního postupu) není vhodné, pokud chcete pravidelně měnit např. teoretické materiály.

0. Přejděte do adresáře BP/Bot/, pokud se zde již nenacházíte.

cd BP/Bot/

1. Sestavte Docker image.

docker build --tag lingebot .

2. Spusťte Docker container.

Pro první spuštění použijte:

docker run --name lingecont -d lingebot

Pro následující spuštění používejte:

docker start lingecont
